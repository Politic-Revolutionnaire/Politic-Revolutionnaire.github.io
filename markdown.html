This is Markdown.

The best programming language ever created.

So, what the hell is markdown? It’s a way of typing documents with quick, universal shorthand. For example, if you’d like a blockquote, like the below:

This is a block quote. Put a random tidbit, some fun fact, etc.

The code is literally just:

> This is a block quote. Put a random tidbit, some fun fact, etc.

And here’s a link.

And the code:

And here's a [link](politic-revolutionnaire.github.io).

Everything is so damn simple, why wouldn’t you use it? And there’s even a direct save to your repo option in many editors, so there’s no reason not to use it.

Just… simple. Markdown.
